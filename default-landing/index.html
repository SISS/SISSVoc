Elda

An implementation of the Linked Data API

RESTful vocabulary-service interface, following the Linked Data API.

URI Templates Available :

  • Query resource identified by a URI

  • List of concept schemes - basic-view

  • List of concept collections - basic-view

  • List of concepts - basic-view

  • List of concepts where the rdfs:label, skos:prefLabel or skos:altLabel contains the quoted text, any language - basic-view

  • List of concepts where the skos:notation is exactly the quoted text - basic-view

    • Format negotiation using HTTP Accept Header :

    Default format is HTML

    Format negotiation can be achieved by appending the request with :

  • .json (should return JSON formatted document) > Example .json
  • .ttl (should return Turtle formatted document) > Example .ttl
  • .rdf (should return RDF/XML formatted document) > Example .rdf
